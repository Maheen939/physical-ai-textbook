# Quick Demo Video Checklist

5-minute setup guide for recording your chatbot demo.

## âš¡ Quick Start (5 Minutes)

### 1. Start Servers (2 min)

```bash
# Terminal 1: Frontend
cd physical-ai-textbook
npm start
# Wait for: http://localhost:3000

# Terminal 2: Backend (optional if no API keys)
npm run chatbot:start
# Wait for: API server started
```

### 2. Prepare Browser (1 min)

- [ ] Open Chrome/Firefox in Incognito mode
- [ ] Go to http://localhost:3000
- [ ] Press F11 for fullscreen
- [ ] Zoom: 100% (Ctrl+0)
- [ ] Close extra tabs

### 3. Set Up Recording (1 min)

**Windows**: Win + G (Game Bar)
**Mac**: Cmd + Shift + 5
**Or**: Download OBS Studio (better quality)

### 4. Prepare Script (1 min)

Copy these queries to Notepad:
```
1. What is ROS 2?
2. How do I add sensors to my robot?
3. Can you show me an example?
```
