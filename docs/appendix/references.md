## Notes on Citations

1. **Preprints**: Papers marked with arXiv links may have been published in peer-reviewed venues after initial preprint release. Check the arXiv page for publication information.

2. **Software Versions**: Many software references (ROS 2, Isaac Sim, etc.) are version-specific. This course primarily uses:
   - ROS 2 Humble Hawksbill
   - NVIDIA Isaac Sim 2023.1.1
   - Gazebo Classic 11 / Gazebo Harmonic

3. **Online Resources**: URLs were verified as of December 2024. Some links may change; use web archives or search engines if broken.

4. **Open Access**: Many robotics papers are available on arXiv.org even if published in paid venues. Always check for preprints.

5. **Code Availability**: Modern papers often provide code repositories. Check supplementary materials or author websites for implementations.
