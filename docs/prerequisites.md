# Prerequisites

Before starting this course, you should have:

## Programming Skills

- **Python 3.10+**: Intermediate level
  - Functions, classes, modules
  - File I/O and error handling
  - Basic numpy and data structures

- **Command Line**: Comfortable with terminal/bash
  - Navigating directories
  - Running scripts
  - Managing packages (pip, apt)

## AI/ML Background

This course assumes you have completed or are familiar with:
- AI Agent development concepts
- Basic machine learning principles
- Large Language Models (LLMs) fundamentals
- Neural networks and deep learning basics

## Mathematics

- **Linear Algebra**: Vectors, matrices, transformations
- **Calculus**: Derivatives (for kinematics/dynamics)
- **Probability**: Basic statistics

## Optional but Helpful

- C++ basics (for performance-critical ROS 2 nodes)
- Git version control
- Docker containers
- Basic computer vision concepts

## Hardware Access

You will need access to:
- **Workstation**: RTX-capable GPU (RTX 4070 Ti or higher) OR cloud alternative
- **Operating System**: Ubuntu 22.04 LTS (dual-boot, WSL2, or VM)
- **Optional**: NVIDIA Jetson Orin Nano for edge deployment

See [Hardware Setup](./hardware-setup.md) for detailed requirements and cloud alternatives.

## Software to Install

Before starting Module 1, ensure you have:
- Ubuntu 22.04 LTS
- Python 3.10+
- ROS 2 (Humble or Iron)
- Gazebo (Classic or Gazebo Sim)
- Git

See [Software Setup](./software-setup.md) for step-by-step installation instructions.

## Time Commitment

- **Course Duration**: 13 weeks
- **Time per Week**: 10-15 hours
- **Total Hours**: 130-195 hours

This course is intensive and hands-on. Plan accordingly!
